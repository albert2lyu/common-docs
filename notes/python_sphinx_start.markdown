## start

安装环境: centos 7.2 64位

`pip install sphinx`


安装完毕之后的设置，逐个设置即可：

`sphinx-quickstart`


完毕之后的目录结构：

```
[root@node128 my_sphinx]# tree
.
├── build
├── make.bat
├── Makefile
└── source
    ├── conf.py
    ├── index.rst
    ├── _static
    └── _templates
```
